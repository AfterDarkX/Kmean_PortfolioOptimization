แบบประเมินความเสี่ยงที่ยอมรับได้ในการลงทุน (Risk Tolerance)

โปรดเลือกคำตอบเพียง 1 ข้อสำหรับคำถาม

1. ทัศนคติของท่านที่มีต่อความเสี่ยงในการลงทุน คืออะไร?

2. ปัจจุบันความมั่นคงและความสามารถในการหารายได้ของท่านตรงกับข้อใด

3. ความเป็นไปได้ที่ท่านจะถอนเเงินลงทุนของท่านในจำนวนที่สูงก่อนกำหนดเวลาการลงทุนที่ตั้งเป้าหมายไว้ เมื่อท่านมีความจำเป็นต้องใช้เงินฉุกเฉิน

4. ท่านคิดว่าอัตราผลตอบแทนที่สมเหตุสมผลจากพอร์ตการลงทุนของท่านเฉลี่ยต่อปีคือเท่าใด

5. ในกรณีเกิดความสูญเสียจากการลงทุน ท่านจะยอมรับการลดค่าลงของเงินทุนในปีหนึ่งปีหนึ่งปีใดมากที่สุดจำนวนเท่าใด?

6. ท่านมีระดับในการออมเงินต่อเดือนในปัจจุบันเป็นจำนวนเท่าใด

7. ในวันหนึ่งวันใดที่ตลาดหุ้นดีดตัวสูงขึ้น ท่านจะ

8. ปัจจุบันท่านมีอายุอยู่ในช่วงใด

9. วัตถุประสงค์หลักในการลงทุนของท่านคือข้อใด

10. ท่านมีประสบการณ์การลงทุนในหลักทรัพย์เสี่ยงมากน้อยเพียงใด
